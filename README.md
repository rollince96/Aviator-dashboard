# Aviator Dashboard

A sleek real-time signal generator dashboard for monitoring and analyzing live data streams.

## Features

- 📊 Real-time data visualization
- 🎯 Signal generation and analysis
- 📈 Interactive charts and graphs
- 🔄 Live updates and streaming data

## Prerequisites

- Node.js (v14 or higher) OR Python (v3.8 or higher)
- npm/yarn OR pip
- Git

## Installation

### Option 1: Node.js/JavaScript Project
```bash
# Clone the repository
git clone https://github.com/rollince96/Aviator-dashboard.git
cd Aviator-dashboard

# Install dependencies
npm install
```

### Option 2: Python Project
```bash
# Clone the repository
git clone https://github.com/rollince96/Aviator-dashboard.git
cd Aviator-dashboard

# Create virtual environment (recommended)
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt
```

## Running the Dashboard

### For Node.js Projects
```bash
npm start
```
The dashboard will be available at `http://localhost:3000`

### For Python Projects
```bash
python app.py
```
The dashboard will be available at `http://localhost:5000`
